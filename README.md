# Cross-Section Dataset from World Bank Data (2022)

## Overview
This repository contains a raw cross-sectional dataset (year 2022) obtained from the World Development Indicators (WDI), World Bank.

The dataset is prepared for econometric regression analysis as part of an individual assignment.

## Data Characteristics
- Year: 2022
- Type: Cross-section
- Observations: 102 countries
- Source: World Development Indicators (World Bank)

## Variables Included
Dependent variable:
- Life expectancy at birth (total)

Independent variables:
- GDP per capita (current US$)
- Infant mortality rate
- Health expenditure (% of GDP)
- Physicians (per 1,000 people)
- Urban population (%)
- Access to basic drinking water (%)

## Data Status
This dataset is currently in raw format as downloaded from the World Bank DataBank.
Further data cleaning, transformation, and regression analysis will be conducted in subsequent stages.

## Source
World Bank. World Development Indicators. https://databank.worldbank.org/
